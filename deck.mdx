# Benefits

<Flex sx={{ alignItems: "center", padding: "20px" }}>
  <Box sx={{ flex: "1 1 50%", marginRight: "40px", textAlign: "center" }}>
    <h3> Content Delivery Network (CDN) </h3>
    <img src={cdn} alt="CDN" />
  </Box>
  <Box sx={{ flex: "1 1 50%" }}>
    <ul>
      <Steps>
        <li>
          <h2>Reduced Complexity</h2>
        </li>
        <li>
          <h2> Lower Costs </h2>
        </li>
        <li>
          <h2> Better Performance </h2>
        </li>
      </Steps>
    </ul>
  </Box>
</Flex>

<Notes>
  <h2>Reduced Complexity </h2>
  <ul>
    <li>
      No need to setup web servers. Instead, we're just serving assets from a
      CDN.
    </li>
    <li>CDN is already globally available, no need to worry about:</li>
    <ul>
      <li>Regions</li>
      <li>Availability zones</li>
    </ul>
  </ul>
  <h2> Lower Costs </h2>
  <ul>
    <li>No servers (EC2. etc) in traditional server model </li>
  </ul>
  <h2>Better Performance</h2>
  <ul>
    <li>Your assets are static, and cached.</li>
    <li>
      Even <strong>BETTER</strong> than serverless, no logic needs to be
      invoked.
    </li>
    <li> Just serve the static assets!</li>
  </ul>
</Notes>
