# Benefits

- Reduced Complexity
- Lower Costs
- Better Performance

<Notes>
  <h1>Reduced Complexity </h1>
  <p>
    No need to setup web servers. Instead, we're just serving assets from a CDN.
  </p>
  <p>
    CDN is already globally available, no need to worry about:
    <ul>
      <li>Regions</li> <li>Availability zones</li>
    </ul>
  </p>
  <h1> Lower Costs </h1>
  <ul>
    <li>No servers (EC2. etc) in traditional server model </li>
  </ul>
  <h1>Better Performance</h1>
  <ul>
    <li>Your assets are static, and cached.</li>
    <li>
      Even <strong>BETTER</strong> than serverless, no logic needs to be
      invoked.
    </li>
    <li> Just serve the static assets!</li>
  </ul>
</Notes>
